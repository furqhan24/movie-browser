# Movie Browser

A responsive movie browsing web application developed using HTML, CSS, and JavaScript. The application allows users to explore movies through category-based navigation while providing an interactive and user-friendly browsing experience.

## Features

* Movie category navigation
* Dynamic content rendering
* User login validation
* Dark and light theme switching
* Local storage integration
* Responsive user interface
* Interactive user experience

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## How It Works

1. Users browse movies through different categories.
2. Movie information is dynamically rendered using JavaScript.
3. Theme preferences are managed through dark and light mode switching.
4. User-related information is stored using browser local storage.
5. The interface updates dynamically based on user interactions.
